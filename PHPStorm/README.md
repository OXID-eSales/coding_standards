PHPStorm
========

##Auto formatter
Copy the file OXID.xml to:

* Mac ~/Library/Preferences/WebIdeXX/config/codestyles/OXID.xml
* Windows <User home>\.WebIdeXX\config\codestyles\OXID.xml
* Linux ~/.WebIdeXX/config/codestyles/OXID.xml
